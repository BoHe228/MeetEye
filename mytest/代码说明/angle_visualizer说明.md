# 鱼眼相机目标角度实时可视化系统说明文档

## 概述

`angle_visualizer.py` 是一个基于 Matplotlib 的实时目标方位可视化系统，通过 WebSocket 接收鱼眼相机的目标检测数据，在两个视图中同时展示目标位置信息。

## 功能特性

### 1. 双视图显示

- **3D 半球体视图**：展示目标的方位角和俯仰角
- **2D 雷达俯视图**：展示目标的水平方位和距离信息

### 2. 实时数据接收

- WebSocket 协议通信
- 自动重连机制
- 支持测试模式（预设固定角度）

### 3. 目标数据支持

- 方位角（Azimuth）
- 俯仰角（Elevation）
- 距离（Distance）
- 支持通过双眼像素距离估算距离

## 界面设计

### 配色方案

采用专业的低饱和度蓝绿色系：

| 元素 | 颜色值 | 说明 |
|------|--------|------|
| 主色 | `#3b82f6` | 蓝色 |
| 辅助色1 | `#10b981` | 翠绿色 |
| 辅助色2 | `#f59e0b` | 琥珀色 |
| 辅助色3 | `#ef4444` | 红色 |
| 背景色 | `#f8fafc` | 浅灰白 |
| 文字色 | `#1e293b` | 深灰 |

### 3D 半球体视图

- **网格线**：12条纬线 + 8条经线，半透明灰色
- **水平面**：蓝色实线 + 淡蓝色填充盘
- **方向标注**：前方、右侧、后方、左侧
- **坐标轴**：X(东)、Y(前)、Z(上)
- **视角**：仰角 28°，方位角 50°
- **显示内容**：
  - 目标点（带黑色边框的彩色圆点）
  - 从中心到目标的箭头连线
  - 信息标注框（ID、方位、俯仰）

### 2D 雷达俯视图

- **同心圆刻度**：1m ~ 5m
- **辐射线**：每 30° 一条，主方向加粗
- **方向标注**：前方、右侧、后方、左侧
- **中心原点**：十字 + 白色内圈
- **显示内容**：
  - 目标点：有距离 = 圆形 / 无距离 = 三角形
  - 从中心到目标的连线
  - 信息标注框（ID、方位、距离）

### 状态栏

- 状态颜色指示：
  - 绿色（#10b981）：已连接 / 测试模式
  - 黄色（#f59e0b）：警告
  - 红色（#ef4444）：错误 / 断开
- 显示内容：连接状态、帧数、目标数量

## 使用方法

### 1. 测试模式

```bash
python angle_visualizer.py --test
```

测试模式下使用预设的 5 个目标：

| ID | 方位角 | 俯仰角 | 双眼像素距离 | 估计距离 |
|----|--------|--------|--------------|----------|
| A | 0° | 0° | 40px | ~1.0m |
| B | 90° | 30° | 20px | ~1.9m |
| C | 180° | 60° | - | 未知 |
| D | 270° | -5° | 7px | ~4.7m |
| E | 45° | 85° | - | 未知 |

### 2. 真实模式

```bash
python angle_visualizer.py ws://192.168.1.100:8000
```

程序会自动补全端点为 `ws://192.168.1.100:8000/ws/inference`

## 数据格式

### WebSocket 接收数据

```json
{
  "targets": {
    "目标ID1": {
      "azimuth": 方位角度数,
      "elevation": 俯仰角度数,
      "distance": 距离(可选, 单位米),
      "eye_pixel_dist": 双眼像素距离(可选)
    },
    "目标ID2": {
      ...
    }
  }
}
```

### 坐标转换

**球坐标 → 笛卡尔坐标**：
- 方位角 0° → Y轴正向（正前方）
- 方位角顺时针增大
- 半径默认 0.85（半球体）

```python
x = radius × cos(el) × sin(az)
y = radius × cos(el) × cos(az)
z = radius × sin(el)
```

## 核心参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `_RADAR_MAX_DIST` | 5.0 | 雷达视图最大显示距离（米）|
| 标注框透明度 | 0.65 | 信息框的背景透明度 |
| DPI | 120 | 渲染分辨率 |
| 刷新帧率 | ~30 FPS | `plt.pause(0.033)` |

## 技术实现

### 1. 线程安全

使用 `threading.Lock()` 保护共享状态，WebSocket 接收线程和 Matplotlib 主线程安全访问数据。

### 2. 动态元素管理

- `_radar_dynamic_artists`：追踪 2D 视图动态元素
- `_sphere_dynamic_artists`：追踪 3D 视图动态元素
- 每次更新前清除上一帧元素

### 3. 3D 箭头实现

继承 `FancyArrowPatch`，自定义 `do_3d_projection()` 方法实现 3D 空间中的箭头绘制。
