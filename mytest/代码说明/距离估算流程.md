# 距离估算流程说明

## 概述

系统通过 YOLO-Pose 检测到的双眼和鼻尖三个关键点，先利用鼻尖的横向偏移估算头部偏转角（Yaw），再将偏转造成的双眼像素距离缩短量修正回来，最后通过经验公式换算出人与摄像头之间的物理距离（米）。

---

## 完整流程图

```
YOLO 关键点
kpt[0]=鼻尖  kpt[1]=左眼  kpt[2]=右眼
         │
         ▼
  提取三点像素坐标
  left_eye (lx, ly)
  right_eye (rx, ry)
  nose (nx, ny)
         │
         ▼
  表观双眼像素距离
  Dapp = dist(left_eye, right_eye)
         │
    ┌────┴──────────────────────────┐
    │                               │
    ▼                               ▼
  鼻尖横向偏移                  偏转角估算
  nose_lateral = nx - (lx+rx)/2   tan(yaw) = nose_lateral / (Dapp/2) / 0.6
                                   yaw_deg = arctan(tan_yaw)
                                        │
                                   yaw > 45° ?
                                   │         │
                                  是         否
                                   │         │
                              返回上帧    透视修正
                              有效距离    Dreal = Dapp / cos(yaw)
                                              │
                                              ▼
                                       经验标定公式
                                  distance = 1 / (K1×Dreal + K2)
                                              │
                                              ▼
                                       物理距离（m）
                                  写入 _last_valid_distance
```

## 各步骤代码说明

### Step 1：提取关键点

**文件**：`utils/distance_estimator.py` → `HeadPoseDistanceEstimator.extract_keypoints()`

```python
# COCO 关键点顺序：kpt[0]=鼻尖, kpt[1]=左眼, kpt[2]=右眼
nose  = (float(kpts[0, 0]), float(kpts[0, 1]))
l_eye = (float(kpts[1, 0]), float(kpts[1, 1]))
r_eye = (float(kpts[2, 0]), float(kpts[2, 1]))
```

若任一关键点置信度低于阈值（默认 0.1），返回 `(None, None, None)`，跳过本帧计算。

---

### Step 2：表观双眼像素距离 Dapp

**文件**：`utils/distance_estimator.py` → `compute_distance()`

```python
Dapp = np.linalg.norm(right_eye - left_eye)
```

Dapp 是当前帧图像中两眼之间的像素距离。当头部正对摄像头时，Dapp 反映真实眼距的完整投影；当头部偏转时，由于透视缩短，Dapp 会小于正脸时的值。

---

### Step 3：估算偏转角 yaw

**文件**：`utils/distance_estimator.py` → `compute_distance()`

```python
# 鼻尖相对于眼睛中点的横向偏移
eye_mid_x   = (left_eye[0] + right_eye[0]) / 2.0
nose_lateral = nose[0] - eye_mid_x

# 归一化（以半眼距为单位，消除分辨率影响）
nose_norm = nose_lateral / (Dapp / 2.0)

# 简化透视模型：nose_norm ≈ 0.6 × tan(yaw)
# 其中 0.6 = 2 × (鼻尖深度 / 眼距) 的人脸均值估计
tan_yaw = abs(nose_norm) / 0.6
yaw_deg = degrees(arctan(tan_yaw))
```

**几何依据**：头部偏转 yaw 角时，鼻尖在图像中的横向位移为 `D_nose × sin(yaw)`，同时眼距缩短为 `W × cos(yaw)`，两者之比消去距离量纲，只与面部比例和偏转角有关。

| 状态 | nose_norm | 估算 yaw |
|------|-----------|---------|
| 正脸 | ≈ 0 | ≈ 0° |
| 轻微偏转 | ≈ 0.3 | ≈ 27° |
| 中度偏转 | ≈ 0.5 | ≈ 40° |
| 严重侧脸 | > 0.6 | > 45°，触发熔断 |

---

### Step 4：熔断机制

**文件**：`utils/distance_estimator.py` → `compute_distance()`

```python
if yaw_deg > self.yaw_fuse_deg:   # 默认阈值 45°
    return self._last_valid_distance
```

偏转角超过阈值时，透视修正误差过大，直接返回上一帧的有效距离。  
每个跟踪目标持有独立的估算器实例，`_last_valid_distance` 在目标存续期间持续保留。

---

### Step 5：透视修正 Dreal

**文件**：`utils/distance_estimator.py` → `compute_distance()`

```python
# 投影关系：Dapp = Dreal × cos(yaw)
# 因此反推真实等效正面像素眼距：
cos_yaw = 1.0 / sqrt(1.0 + tan_yaw ** 2)
Dreal   = Dapp / cos_yaw          # Dreal ≥ Dapp
```

Dreal 是"如果这个人正对摄像头时，双眼之间应有的像素距离"，消除了偏转造成的缩短，使后续经验公式能正确使用。

---

### Step 6：经验公式换算物理距离

**文件**：`utils/distance_estimator.py` → `estimate_distance_from_eyes()`

```python
K1 = 0.024030
K2 = 0.044812

distance = 1.0 / (K1 × Dreal + K2)    # 单位：米
```

公式形式来源于相似三角形原理：

```
distance ∝ IPD_real / D_pixel
```

K1、K2 是在固定摄像头、固定人群（IPD 均值 ≈ 63mm）条件下实测标定的，隐含了焦距和平均眼距两个参数。公式假设所有人眼距均为人群均值，个体差异（约 ±15%）会带来对应的距离估算误差。

---

### Step 7：写入缓存，打包发送

**文件**：`main_GPU_webui.py` → `_build_inference_json()`

```python
# 每个 track_id 对应一个独立的估算器
estimator = _distance_estimators.get(tid)
if estimator is None:
    estimator = HeadPoseDistanceEstimator()
    _distance_estimators[tid] = estimator

l_eye, r_eye, nose = HeadPoseDistanceEstimator.extract_keypoints(kpts)
if l_eye is not None:
    distance = estimator.compute_distance(l_eye, r_eye, nose)
else:
    distance = estimator.last_valid_distance    # 关键点不可信时沿用上帧

# 帧结束后清理已消失目标的估算器
stale = [tid for tid in _distance_estimators if tid not in current_tids]
for tid in stale:
    del _distance_estimators[tid]
```

## 误差来源汇总

| 误差来源 | 量级 | 说明 |
|---------|------|------|
| 个体 IPD 差异 | ±15% | 公式假设所有人眼距 63mm |
| 鼻深比例假定 | 小角度可忽略 | `_NOSE_SCALE = 0.6` 为人脸均值，影响偏转角精度 |
| 关键点检测误差 | 随置信度变化 | 低置信度帧被过滤，沿用上帧距离 |
| 严重侧脸熔断 | 直接沿用上帧 | yaw > 45° 时停止更新 |
