# 全景鱼眼 YOLO 姿态检测系统 - 总览

## 系统架构

```
鱼眼摄像头/视频输入
    ↓
相机展开（CameraProcessor）
    ↓
全景切片 + 并行检测（PanoramaSlicer）
    ↓
切片检测结果合并 + 环绕去重（PanoramaSlicer）
    ↓
ReID特征提取
    ↓
BoT-SORT 跟踪（边界穿越匹配）
    ↓
角度计算
    ↓
结果显示/保存
```

---

## 核心模块说明

### 1. PanoramaSlicer（同一帧内去重）
- **作用**: 把全景切成3片，分别检测，然后合并去重
- **关键功能**:
  - 环绕边界去重（slice 0 & slice 2，用 ReID 特征）
  - 相邻边界去重（slice 0&1, slice 1&2，用 IoU + ReID）
- **解决的问题**: 同一目标横跨左右边界时被检测成两个

### 2. BoT_SORTTracker（跨帧跟踪）
- **作用**: 给检测结果分配稳定的 ID
- **关键功能**:
  - 高/低置信度两级匹配（ByteTrack 策略）
  - IoU + ReID 特征融合（DeepSORT 策略）
  - 边界穿越匹配（全景特有）
- **解决的问题**:
  - 目标被遮挡后再出现 ID 保持一致
  - 目标穿越全景左右边界 ID 保持一致

### 3. BoundaryIDMatcher（边界穿越匹配）
- **作用**: 当目标从一侧消失、从另一侧出现时，复用原来的 ID
- **触发条件**:
  - 目标在边界区域消失 → 记录特征和 ID
  - 目标在边界区域（同侧或对侧）出现 → 匹配特征
  - 特征相似度 > 阈值 → 复用 ID

## 关键数据流

### 同一帧内的流程（PanoramaSlicer）
```
全景图
    ↓
切3片（slice 0, 1, 2）
    ↓
每个切片单独 YOLO 检测
    ↓
检测结果转换回全景坐标
    ↓
提取 ReID 特征
    ↓
去重合并：
    ├─ slice 0 & slice 2: 特征匹配 → 环绕去重
    └─ slice 0&1, slice 1&2: IoU + 特征 → 相邻去重
    ↓
合并后的检测列表（含特征）
```

### 跨帧的流程（BoT_SORTTracker）
```
输入：当前帧检测 + 上一帧轨迹
    ↓
第一级匹配：高置信度检测 + 所有轨迹（IoU + 特征）
    ↓
第二级匹配：低置信度检测 + 未匹配轨迹（仅 IoU）
    ↓
边界穿越匹配：
    ├─ 记录消失在边界的目标
    └─ 新目标与消失目标特征匹配
    ↓
创建新轨迹
    ↓
输出：带 ID 的检测结果
```

## 参数调优指南

### 如果边界穿越 ID 变了
1. **首先检查**: `enable_boundary_matching=True`
2. **调大**: `boundary_time_window`（例如 60 → 90）
3. **调小**: `boundary_similarity_thresh`（例如 0.3 → 0.2）
4. **调大**: `boundary_margin`（例如 0.1 → 0.15）
5. **确认**: `with_reid=True`

### 如果被遮挡后 ID 变了
1. **调大**: `track_buffer`（例如 60 → 120）
2. **调小**: `track_low_thresh`（例如 0.3 → 0.1）
3. **调小**: `appearance_thresh`（例如 0.25 → 0.15）
4. **调小**: `match_thresh`（例如 0.7 → 0.3）

### 如果误匹配太多
1. **调大**: `match_thresh`（例如 0.3 → 0.5）
2. **调大**: `appearance_thresh`（例如 0.15 → 0.3）
3. **调大**: `track_high_thresh`（例如 0.5 → 0.6）

### 如果漏检太多
1. **调小**: `track_high_thresh`（例如 0.5 → 0.4）
2. **调小**: `new_track_thresh`（例如 0.3 → 0.25）
3. **调大**: `PanoramaSlicer.overlap_ratio`（例如 0.1 → 0.15）

## 日志说明

### DetectionLogger
程序结束时会保存 JSON 日志到 `yolo_pose_output/detection_log_*.json`，包含：
- YOLO 模型名
- OSNet 模型名
- 每帧的 YOLO 检测 IDs
- 每帧的最终跟踪 IDs
- 每帧的 YOLO 检测置信度
- 每帧的最终跟踪置信度
- 统计信息：
  - 总帧数
  - YOLO 检测到目标的帧占比
  - 最终跟踪到目标的帧占比
  - 各置信度区间的检测帧占比

## 常见问题 FAQ

### Q: 我的边界穿越功能没有生效？
A: 检查:
1. `enable_boundary_matching=True`
2. `with_reid=True`
3. OSNet 模型加载成功（没有初始化失败的打印）
4. 查看边界调试输出是否有 `[消失目标]`、`[新目标]` 打印

### Q: 我觉得 ReID 特征匹配不工作？
A: 检查:
1. `feature_extractor` 初始化成功了吗？
2. `PanoramaSlicer` 是否正确提取了特征？
3. 检测框是否太小？太小的话特征提取效果不好

### Q: 如何查看当前是什么配置在运行？
A: 在 `main.py` 中 `BoT_SORTTracker` 初始化处加打印，或者查看本目录下的参数说明文档。

### Q: 边界区域在哪里可以看到？
A: 开启 `boundary_debug=True`，画面上会画出左右边界的区域。

### Q: 我想保存视频给别人看，应该用什么参数？
A:
```
python mytest/main.py --video-path xxx.mp4 --save-video --use-dual-windows
```
